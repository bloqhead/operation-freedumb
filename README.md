## What Is This

A silly satirical web app mocking the seemingly random and unverifiable claims of wars being ended. Roll a die, get a number, receive a list of completely fabricated conflicts between countries that definitely exist (spelled correctly or not) — along with a very authentic-sounding presidential quote.

**All wars, countries, quotes, and geopolitical events are entirely fabricated.** Any resemblance to actual foreign policy is, frankly, concerning.

## Tech Stack

- [Vue 3](https://vuejs.org/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [VueUse](https://vueuse.org/)

## Getting Started


## Disclaimer

This is satire. The First Amendment is, for now, still a thing. All names, places, quotes, and conflicts are made up. The author does not actually believe any wars were or were not ended. The author does believe Flargistan is a real place.
